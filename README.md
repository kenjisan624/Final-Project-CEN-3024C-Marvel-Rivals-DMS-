# CEN-3024C-Marvel-Rivals-DMS-

This is a DMS which stands for Data Management System in the style of the game Marvel Rivals hero base online team player game; this project has been developed for the
Class taken at Valencia College || Software Development I. 

**Description of the project: **
This program incorporate  Logic - CRUD operation ( Create, Remove, Update, and Display) + Graphic Interdace + Database.
Allowing user to either use Logic interface or the all the functions equipped GUI where user will be able to manage .db files of the heroes sample based on Season 0 of the Marvel Rivals. 
As previously mentioned it can add, delete, edit, and view hero records with a nice graphic interface. 
It has incorporated a custom method which generate a PDF on ascending order of win rate of the heroes. 


# Pre - Requisites:
- IDE 
- Java SDK
- SQLite
